- wu.Rdata

#### Dados de voos

- vra2.Rdata - (2003 a 2005)
- vra3.Rdata - (2006 a 2008)
- vra4.Rdata - (2009 a 2011)
- vra5.Rdata - (2012 a 2014)
- vra6.Rdata - (2015 a 2017)
